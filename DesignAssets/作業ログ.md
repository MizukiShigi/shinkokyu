# 森呼吸 — デザイン作業ログ

最終更新: 2026-06-11

---

## 概要
ブリーフ「森呼吸(しんこきゅう)」に基づくデザイン一式。
成果物は単一ファイル `森呼吸 Design.dc.html` にまとめている(設計意図 / ロゴ / ワイヤー / 高精細 / 動くプロトタイプ / モーション仕様)。

---

## 変更履歴

### v1 — 初版
- カラーを再設計: 元の #0F6E56 系から、彩度を落とし青みを引いた単一色相パレットへ(墨 / 深林 / 松 / 苔 / 若葉 / 霞 / 和紙)。アクセント色は持たない。
- 書体: 見出し=しっぽり明朝 / UI=Zen角ゴシック New。
- ロゴ: 案B「波紋の樹冠」を3案展開。**B-2「めぐる波紋」を推奨**(波紋が下で開き、幹が中心を通って地面へ届く)。
- 3画面の高精細デザイン(スプラッシュ / ホーム / おかえり)+ セッション写真3パターン検証(明るい昼 / 夕暮れ / 暗い森)。
- インタラクティブプロトタイプ(タップ→呼吸4秒/6秒→自動でおかえり、デモは30秒短縮)。
- モーション仕様(呼吸円キーフレーム / 画面遷移タイミング)。

### v2 — レビュー反映(コメント対応)
- **ホームから「きょうの森」を削除。** 場所表示と日替わり前提をやめ、ホームを「ロゴ / 円ボタン / 今週の回数」の3要素構成に。ボタンが画面の重心になる潔い構成へ。
- **おかえり画面の文言を変更。**「おかえりなさい」→「おつかれさまでした」、添え書き「3分間、森にいました」→「3分間、深呼吸しました」(森にいた、という比喩をやめ平易に)。
- 設計意図 / ワイヤー / 高精細 / プロトタイプの説明文を上記に整合。

### v3 — 実写真の適用
- ご提供の9枚から3枚を時間帯に割り当ててセッション画面に適用(高精細3パターン + プロトタイプ):
  - 明るい昼 / 奥多摩・倉沢のせせらぎ → `assets/photos/kurasawa-day.jpg` (anders-ipsen)
  - 夕暮れ / 上高地・明神 → `assets/photos/kamikochi-dusk.jpg` (mahiru-dake)
  - 夜 / 屋久島・白谷 → `assets/photos/yakushima-night.jpg` (max-saeling)
- 予備: `assets/photos/alt-autumn-stream.jpg` (zhaoli-jin / 紅葉の渓流) を控えに保持。
- スクリム(上下の暗い帯)は据え置きで、明るい滝つぼ〜暗い星空まで文字可読性が成立することを確認。

## アイコン書き出し(assets/)
- `icon-mark.svg` — 透過の単体マーク(B-2)
- `app-icon-light-1024.svg` / `app-icon-dark-1024.svg` — フルマーク版のアプリアイコン(SVG)
- `app-icon-light-small.svg` — 小サイズ用の簡略版(外周の波紋を1本落とす)
- `app-icon/light/icon-{1024,180,167,152,120,87,80,76,60,58,40,29,20}.png` — iOS各サイズ(58px未満は簡略版で書き出し)
- `app-icon/dark/icon-{1024,180,120}.png` — ダーク地反転版

カラー値: マーク #306450(松) / ライト地 #F8FAF8→#E2EBE6 / ダーク地 #1B3D31→#15281F / 反転マーク #E3F1EB

> 備考: iOSのアプリアイコンは角丸をシステムが自動マスクするため、書き出しは正方形・角丸なし。App Store用は light/icon-1024.png。

### v4 — 文言・ロゴ確定
- ホームの開始ボタンを「森呼吸する」→「はじめる」に変更(動詞ひとつでシンプルに / 字間を広げて調整)。ワイヤー・高精細・プロトタイプ・操作説明に整合。
- **ロゴは B-2「めぐる波紋」で確定。** アプリアイコン書き出し・全画面・スプラッシュはすべてB-2で統一済み。

## 保留 / 次の判断待ち
- おかえり画面の文言は他候補あり(「ひと息つきました」「いい呼吸でした」「ゆっくり、戻りましょう」)。確定で差し替え。
- セッション画面上部の場所名表示(奥多摩・倉沢…)は現状維持。ホームに合わせて外すかは要判断。
- 実写真・環境音は未収録。提供後に3パターン検証を実素材で再実行。
- ダーク地ホーム(夜間起動時の反転)は今のパレットでそのまま展開可能。
