# 森呼吸 (しんこきゅう)

3分、森で深呼吸するだけのiOSアプリ。応急処置である / 選択肢ゼロ / 何も要求しない。

デザイン一式は `DesignAssets/作業ログ.md` と Claude Design のハンドオフバンドル（`森呼吸 Design.dc.html`）に基づく。

## 構成

- **SwiftUI** (iOS 17+) — 画面3枚 + スプラッシュ、遷移はすべてクロスフェード
- **AVAudioPlayer** — 環境音(CC0フィールドレコーディング)。開始2.0sフェードイン / 終了は鐘 + 6.0sフェードアウト。サイレントスイッチONでも鳴る(`.playback`)
- **Core Haptics** — 吸う=強度0→0.6を4秒 / 吐く=0.6→0を8秒。目を閉じても呼吸を追える
- **バックエンドなし** — 通信ゼロ・権限ダイアログゼロ。記録は「今週の回数」のみ(UserDefaults)

```
Shinkokyu/
  App/        ShinkokyuApp, RootView (画面遷移と配線)
  Model/      SessionEngine (呼吸の状態機械) / AudioController / HapticsController
              ForestScene (14景のカタログ) / Palette (oklch→sRGB変換済み) / WeeklyCounter
  Views/      Home / Session / Okaeri / Splash / RippleMark (ロゴB-2)
  Resources/  Photos (HEIC ×14) / Sounds (AAC ×9)
  Fonts/      しっぽり明朝 + Zen角ゴシック New (OFL)
```

## セッションの仕様

- **180秒 = 12秒呼吸ループ(吸4秒/吐8秒・1:2のリラックス比) × 15サイクル**。タイマーと呼吸が同期していれば3:00ちょうどが15回目の吐く終点
- 呼吸円内の秒表示はカウントダウン(吸4,3,2,1 / 吐8,7,…,1)
- 「吸う/吐く」+秒は最初の3サイクル(36秒)だけ表示し、吐き切ったところで2.5秒かけて溶ける。以降は円の拡縮と明度変化だけで呼吸を示す(リズムを覚えたら文字を引く)。場所名と残り時間は最後まで残す
- **開始前の導入**(計約9.5秒): 景色と環境音が先に立ち上がり、メッセージがゆっくり現れて消える→「3・2・1」(明朝40pt・明滅)。3と2は短く、**1だけ1.5秒かけてゆっくり溶けて、消えきった瞬間に最初の吸うが始まる**。導入中は一時停止ボタン非表示
- 呼吸円の拡縮は 1.00⇄1.28。デザイン原案の1.40から縮小 — 吸う4秒は吐く8秒の2倍速の移動になるため、幅を縮めて吸うのピーク速度を吐くの体感に揃えた(吸うはほぼ等速カーブ、吐くはease-in-out)
- **写真は30秒ごとに同じ時間帯の景へ2秒クロスフェード**(場所名も追従)。環境音は最初の景のまま継続。Ken Burnsは写真ごとに1.00→1.05/linearで掛け直し(デモ時は10秒間隔)
- 終了条件は「180秒経過後、**現在の吐く動作が終わった時点**」— 呼吸の途中でぶつ切りにしない
- 終了の瞬間: 鐘(合成したりん風の音、9秒減衰)が鳴り、その余韻の下で環境音が6秒かけて消える
- 一時停止: 環境音・Hapticsも止まる。再開は「吸う」の頭から。中断したフェーズ分の時間は数え直し
- 一時停止中にだけ「おわる」が現れる(唯一の中断手段。回数は数えない)
- **バックグラウンド再生対応**(UIBackgroundModes: audio): ロックしてもセッションは進み、環境音と鐘は鳴り続ける。Hapticsと画面ガイドはiOSの仕様によりバックグラウンドでは停止。着信(オーディオ中断)時は自動ポーズ / セッション中は画面自動ロック無効
- Reduce Motion設定時は拡縮をやめ、円の充填の明滅だけで呼吸を示す

## 景色の選ばれ方(選択肢ゼロ)

開始のたびに全20景からランダムで1景を選び、30秒ごとに(残り全景をシャッフルした順で)次の景へ移る。ユーザーは選ばない。
※ホームの昼/夜テーマだけは時刻(夜19-5時)で切り替わる。景色の選択自体は時間帯に依存しない。

| 時間帯 | 景色 |
|---|---|
| 昼(15景) | 華厳の滝・嵐山竹林・保津川・醍醐寺・苔の庭・千畳敷カール・杉の森・美瑛青い池・上高地梓川・石神井公園×3・井の頭公園・どっこ沼・鳥兜山 |
| 夕(3景) | 那智の滝・駒ヶ根・箱根芦ノ湖 |
| 夜(2景) | 星の森・富士山湖畔 |

## ビルド

1. Xcode 16+ で `Shinkokyu.xcodeproj` を開く
2. シミュレータ実行には iOSランタイムが必要: Xcode → Settings → Components(または `xcodebuild -downloadPlatform iOS`)
3. 実機で動かす場合は Signing & Capabilities で自分のTeamを選択
4. **Hapticsはシミュレータでは動かない。** 体感確認は必ず実機で

### デバッグ用起動引数 (Edit Scheme → Arguments)

- `-demo30` — セッションを30秒(3サイクル)に短縮。デザインプロトタイプの30秒デモと同じ
- `-autoStartSession` — 起動後ホームを経由して自動でセッション開始(UI検証用)
