# リリース準備チェックリスト

最終更新: 2026-06-11。現状: シミュレータで全機能動作確認済み / 実機未検証 / App Store Connect未着手。

## Phase 1 — アカウントと実機 (最初にやる・待ち時間があるため)

- [ ] **Apple Developer Program に加入** ($99/年)。承認に1〜2日かかることがあるので最初に着手
- [ ] Xcode → Signing & Capabilities で Team を設定 (今は `DEVELOPMENT_TEAM = ""`)
- [ ] Bundle ID `com.shigihara.shinkokyu` を確定 (変えるなら今が最後のチャンス)
- [ ] **実機にインストールして体感検証**:
  - [ ] Haptics — 吸う(0→0.6/4秒)・吐く(0.6→0/8秒)が気持ちいいか。**シミュレータでは一切検証できていない最重要項目**
  - [ ] 音 — スピーカー/イヤホン両方で環境音と鐘のバランス
  - [ ] サイレントスイッチONでも音が鳴ること
  - [ ] ロック中のバックグラウンド再生 → 3分後に鐘 → 復帰でおかえり画面
  - [ ] 着信(オーディオ中断)→ 自動ポーズ → 復帰
  - [ ] 小さい画面 (iPhone SE系 375×667pt) でレイアウト崩れがないか — ホームの円296pt+ボタン230ptは要確認
