## 実機確認ガイド (初回インストールの手順)

### 1. Apple Developer Program に加入する

1. https://developer.apple.com/programs/enroll/ にアクセス
2. 個人として登録 ($99/年、クレカ払い)
3. 承認完了メールを待つ (最大1〜2日)

### 2. Xcode で Team を設定する

1. `Shinkokyu.xcodeproj` を開く
2. Navigator → `Shinkokyu` ターゲット → **Signing & Capabilities** タブ
3. Team: 自分の Apple ID を選択
4. `ShinkokyuWidget` ターゲットでも同じ操作
5. Bundle Identifier が `com.shigihara.shinkokyu` になっていることを確認

   > App Group (`group.com.shigihara.shinkokyu`) はこの時点でXcodeが自動登録する。「Add App Group」の確認ダイアログが出たら **許可**。

### 3. 実機にインストールする

1. iPhoneをUSBでつなぐ (「このコンピュータを信頼する」で信頼)
2. Xcodeのデバイス選択で実機を選ぶ
3. **⌘R** でビルド&インストール
4. 初回は「設定 → 一般 → VPNとデバイス管理」でデベロッパを信頼する

### 4. 実機で確認するポイント

#### Haptics ← **最重要・シミュレータ不可**
- セッション開始後、画面を持ってスマホを感じながら呼吸に合わせてHapticsが変化するか
  - 吸う4秒: 強さが 0→0.6 に徐々に増す
  - 吐く8秒: 強さが 0.6→0 に徐々に弱まる
- 違和感があれば `HapticsController.swift` のパラメータ調整が必要

#### 音
- **スピーカーで**: 環境音と鐘のバランス (鐘が8dB大きいのが自然に聞こえるか)
- **イヤホンで**: 立体感があるか
- **サイレントスイッチON (マナーモード) のまま** でアプリを起動してセッション開始 → **音が鳴ること**を確認 (AVAudioSession `.playback` カテゴリの動作確認)

#### バックグラウンド再生 & ロック画面
1. セッションを開始して音が出ていることを確認
2. ホームボタン(またはスワイプ)でアプリを閉じる → **音が止まらないこと**
3. スリープボタンで画面をロック → **音が止まらないこと**
4. ロック画面に **「場所名 — 森呼吸」** のメディアカードが表示されること
5. ロック画面の **一時停止ボタン** を押す → 音と円アニメが止まること
6. 3分後 (または `-demo30` ビルドで30秒後) → **鐘が鳴り**、環境音がフェードアウト
7. アプリに戻ると「おかえり」画面

#### コントロールセンター
- セッション中にコントロールセンターを開く (右上から下スワイプ)
- **「森呼吸」のメディアカード**が表示されること
- カードの一時停止ボタンで一時停止できること

#### 着信割り込み
- セッション中に誰かに電話してもらう (または別の端末から)
- 着信で音楽が自動ポーズ → 通話終了後に音楽が自動再開すること

#### レイアウト (iPhone SE / 小画面)
- iPhone SE (375×667pt) をお持ちであればシミュレータで確認
  - ホーム画面: 円296ptとボタン230ptが収まるか
  - セッション画面: 場所名が Dynamic Island / ステータスバーと被らないか

### 5. ウィジェットの確認

1. ホーム画面を長押し → 左上の **「+」** ボタン
2. 「森呼吸」を検索 → **小サイズ** を選択して追加
3. ウィジェットに「今週 0回」と表示されること
4. セッションを1回完走 → ウィジェットが「今週 1回」に更新されること
