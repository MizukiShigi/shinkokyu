# リリース準備チェックリスト

最終更新: 2026-06-11。現状: シミュレータで全機能動作確認済み / 実機未検証 / App Store Connect未着手。

## Phase 1 — アカウントと実機 (最初にやる・待ち時間があるため)

- [ ] **Apple Developer Program に加入** ($99/年)。承認に1〜2日かかることがあるので最初に着手
- [ ] Xcode → Signing & Capabilities で Team を設定 (今は `DEVELOPMENT_TEAM = ""`)
- [ ] Bundle ID `com.shigihara.shinkokyu` を確定 (変えるなら今が最後のチャンス)
- [ ] **実機にインストールして体感検証**:
  - [ ] Haptics — 吸う(0→0.6/4秒)・吐く(0.6→0/8秒)が気持ちいいか。**シミュレータでは一切検証できていない最重要項目**
  - [ ] 音 — スピーカー/イヤホン両方で環境音と鐘のバランス
  - [ ] サイレントスイッチONでも音が鳴ること
  - [ ] ロック中のバックグラウンド再生 → 3分後に鐘 → 復帰でおかえり画面
  - [ ] 着信(オーディオ中断)→ 自動ポーズ → 復帰
  - [ ] 小さい画面 (iPhone SE系 375×667pt) でレイアウト崩れがないか — ホームの円296pt+ボタン230ptは要確認

## Phase 2 — アプリ本体の残タスク

優先度高 (リリース前にやるべき):

- [ ] **夜のダークホーム** — 夜バケット時にホームを深林地に反転 (デザイン文書に方針あり)。寝る前利用で白画面は眩しい
- [ ] **Now Playing対応** (MPNowPlayingInfoCenter / MPRemoteCommandCenter) — ロック画面に「森呼吸 — 場所名」と一時停止を表示。バックグラウンド再生の審査正当性の裏付けにもなる
- [ ] **VoiceOverラベル** — 一時停止/再開ボタン(図形のみ)に accessibilityLabel

任意 (後回し可):

- [ ] 鐘を実録音のりん/梵鐘 (CC0) に差し替え (現状は合成音)
- [ ] 場所名3つの実名化: 「苔の庭」「杉の森」「星の森」は撮影地不明のため仮名 (`ForestScene.swift`)
- [ ] SessionEngine のユニットテスト (吐き切り終了 / ポーズ復帰 / カウントダウン)

## Phase 3 — App Store Connect 提出物

- [ ] **アプリ名の確保**: 「森呼吸」が取られていないか App Store Connect で確認 (取られていたら「森呼吸 - 3分の森林浴」等)
- [ ] サブタイトル (30字): 例「3分、森で深呼吸するだけ」
- [ ] 説明文 — ⚠️ 「治療」「効果」など医療的表現はNG。「リラックス」「ひと息」はOK
- [ ] キーワード (100字): 呼吸,深呼吸,瞑想,リラックス,森林浴,睡眠,マインドフルネス,自然音 など
- [ ] **スクリーンショット**: 6.9インチ (iPhone 17 Pro Max シミュレータで撮影可) 最低3枚。ホーム/セッション(昼・夜)/おかえり
- [ ] **プライバシーポリシーURL** — 収集ゼロでもURL必須。GitHub Pages か リポジトリ内 md へのリンクで可。内容は「本アプリは通信を行わず、いかなるデータも収集しません」レベルで足りる
- [ ] App Privacy (プライバシーラベル): 「データを収集しない」を選択
- [ ] カテゴリ: ヘルスケア/フィットネス (サブ: ライフスタイル)
- [ ] 年齢区分: 4+
- [ ] **審査メモ (App Review Information)**: バックグラウンドオーディオの用途を一文で説明 —「呼吸セッション中(3分間)の環境音再生のために UIBackgroundModes: audio を使用。画面ロック中もセッションの環境音と終了の鐘を再生する」
